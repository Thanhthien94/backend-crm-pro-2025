# CRM Backend API

A comprehensive Customer Relationship Management (CRM) API built with Node.js, Express, TypeScript, and MongoDB.

## Features

- Multi-tenant architecture
- User authentication and authorization
- Customer, Deal, and Task management
- Custom fields for entities
- Webhooks for integration with external systems
- API Key authentication for external access
- Export and import functionality
- Analytics and reporting
- Swagger API documentation

## Prerequisites

- Node.js 16+
- MongoDB
- npm or yarn
